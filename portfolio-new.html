LyricLingo

Live Demo GitHub

A full-stack web application that helps users learn new languages through music. By integrating with Spotify, LyricLingo allows users to log songs, view translated lyrics, create flashcards, and analyze song sentiment.

Key Features

  • Spotify integration to log currently playing songs
  • Multi-language lyrics translation with DeepL API
  • Automatic flashcard generation for language learning
  • Lyric sentiment analysis with Hugging Face AI
  • Personal song history with saved translations

Technical Achievements

  • Redis caching reduced translation API costs by 40%
  • Lyric deduplication algorithm for efficient processing
  • RESTful API architecture with JWT authentication
  • Responsive design with Framer Motion animations
  • 82% accuracy in NLP sentiment analysis
